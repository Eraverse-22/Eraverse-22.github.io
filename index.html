Arduino · FSM Control · Bluetooth · Servo Integration
// About the Project

A 2-link robotic arm (20cm + 10cm) built as a university engineering project. The system uses Finite State Machine (FSM) logic to handle different operational states — idle, receiving commands, executing movement, and grip control.

Controlled wirelessly via Bluetooth (HC-05). My role focused on electrical system design, control logic implementation, and servo integration including torque management for stable pick-and-place operations.

Electrical system design and wiring
FSM control logic (C++ / Arduino)
